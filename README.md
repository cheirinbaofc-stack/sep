# SEP Projetos 3D - Landing Page

## Projeto Completo

Landing page profissional para SEP Projetos 3D com design glassmorphism e tema escuro premium.

### Tecnologias Utilizadas
- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- shadcn/ui

### Recursos Implementados
- ✓ Design glassmorphism com tema escuro
- ✓ Partículas animadas no background
- ✓ Navegação fixa com blur effect
- ✓ Menu hamburger animado para mobile
- ✓ Hero section com CTA buttons magnéticos
- ✓ Cards de serviços com efeito shine border
- ✓ Timeline do processo
- ✓ Cards de equipe (Samanta e Pablo)
- ✓ Formulário de contato com integração WhatsApp
- ✓ Animações scroll progress
- ✓ Footer completo
- ✓ Totalmente responsivo

### Como Executar Localmente

```bash
npm install
npm run dev
```

Acesse http://localhost:3000

### Como Fazer Deploy

#### Opção 1: Vercel (Recomendado)
1. Crie uma conta em https://vercel.com
2. Conecte seu GitHub
3. Importe o repositório
4. Deploy automático

#### Opção 2: Netlify
1. Crie uma conta em https://netlify.com
2. Faça upload da pasta `dist` ou conecte o repositório
3. Deploy

#### Opção 3: Manual
```bash
npm run build
# A pasta dist conterá os arquivos estáticos
```
